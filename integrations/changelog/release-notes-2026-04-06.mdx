## Release Notes — April 6, 2026
> 8 changes · EkaAgents, EMR, Platform

### 🤖 EkaAgents

- Added — ABHA login and profile tools in the MCP SDK. Agents can now initiate ABHA login via mobile, Aadhaar, ABHA number, or PHR address, verify OTP, select from multiple profiles, and retrieve the ABHA card — all within a single agent conversation. [View docs](/ai-tools/mcp-server/supported-tools)

- Improved — `_meta` field forwarding in `doctor_discovery` and `doctor_availability` MCP tools. MCP clients can now pass additional metadata (UI hints, session context) when invoking these tools. [View docs](/ai-tools/mcp-server/supported-tools)

- Improved — MCP Server documentation restructured into separate [Remote MCP](/ai-tools/mcp-server/remote-mcp) and [Local MCP SDK](/ai-tools/mcp-server/local-mcp) guides with per-client setup instructions, a comparison table, and clearer onboarding for both OAuth and API-credential flows. [View docs](/ai-tools/mcp-server/introduction)

- Improved — MedAssist iOS background session handling. Chat sessions now reconnect automatically after app backgrounding, preventing conversation loss mid-interaction.

- Fixed — MedAssist iOS WebSocket error handling. Session timeouts and connection drops are now surfaced with clear error messages instead of failing silently.

### 📋 EMR

- Improved — Prescription PDFs now include pachymetry, Amsler grid, and contact lens examination tables for ophthalmology visits, expanding printed eye-care data. [View docs](/integrations/core-emr/prescription)

### ⚙️ Platform & Infrastructure

- Fixed — Changelog Gmail subscription links now resolve correctly.
- Fixed — GitHub links on documentation pages now point to the correct renamed repositories.
