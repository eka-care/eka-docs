---
openapi: put /assessment/api/v1/continue/{hash}/{qid}
---
<Note>
The `hash` in the URL is the unique identifier for the assessment which was provided in the response of init assessment API. The `qid` in the URL is the unique identifier for the question starting from 0 (first question).
</Note>

<Tip>
Refer to the [list component](/api-reference/general-tools/assessment/component-info/list-all-components) documentation for different question compoent response format. 
</Tip>
