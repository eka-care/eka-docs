### Here's a short description of each component:
<Tabs>
  <Tab title="Autosuggest(I-ASTG)">
    - **Description**: Provides real-time suggestions as the user types in the input field.
    - **Properties**:
        * `id`: question ID.
        * `comp`: Component code.
        * `url`: URL for autosuggest suggestions.
            1. `query_params`: Additional query parameters.
        * `text`: Question text.
        * `tip`: Additional information.
        * `resp`: List of responses.
        * `details`: Additional details.
            1. `static sections`: List of sections.
            2. `heading`: Section title.
            3. `items`: List of choices.
                - `id`: choice ID.
                - `label`: choice label.
        * `is_mandantory`: True/False to show whether the question is skippable or not.
        * `progress`: Progress percentage.
    - **Example** >-
        ```json
        {
            "questions": [
        ```

  <Tab title="Radio Group(I-RADG)">
    - **Description**: Allows selection of choices and sub-choices from a list.
    - **Properties**:
        * `id`: question ID.
        * `comp`: Component code.
        * `text`: Question text.
        * `tip` : Additional information.
        * `resp`: List of responses.
            - `id`: response choice ID.
            - `label`: response label.
            - `choices`: List of choices.
                - `id`: choice ID. (p/a/u for present/absent/unknown)
                - `label`: choice label. (Yes/No/Unknown)
        * `is_mandantory`: True/False to show whether the question is skippable or not.
        * `progress`: Progress percentage.
    - **Examples** >-
        ```json
        {
            "type": "single",
            "questions": [
                {
                    "id": 8,
                    "text": "Is your fever relieved by",
                    "option_type": null,
                    "icon": null,
                    "comp": "I-RADG",
                    "tip": "Select one answer in each row.",
                    "resp": [
                        {
        ```
