---
title: Upload Record
method: GET
---
Upload an image or PDF file that meets the following specifications. Upon successful upload, a document_id will be returned.
## Accepted Document Types:
    * PDF documents
    * Images
        * Preferred DPI: 150 or higher
        * Images should be clear (no blurring)
        * Avoid images with excessive non-linear distortion or rotation

## Query Parameters
Specifies the type of document being uploaded.
The following values are allowed:
    * **ps**: Prescription
    * **lr**: Lab Report
    * **ot**: Other
    * **ds**: Discharge Summary
    * **vc**: Vaccine Certificate
    * **in**: Insurance
    * **iv**: Invoice
    * **sc**: Scan


## Task Parameters
Specifies the type of tasks need done in report.
The following values are allowed:
    * **smart**: Smart Report
    * **pii**: Personal Identification Information
    * **classification**: classify report type
