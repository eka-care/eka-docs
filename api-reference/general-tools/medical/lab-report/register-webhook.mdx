---
title: "Register Webhook"
---

To integrate webhook for Medical Document Parsing, register for a webhook [**here**](/api-reference/connect/webhooks/register-webhook) with the following scopes:

```json
{
    "event_names": [
        "rec.pii" // parsing
        "rec.srp" // smart report parsing
        "rec.clf" // classification
    ],
}
```

For more information on registering, retrieving or deleting webhooks, see the [**webhook section**](/api-reference/connect/webhooks/getting-started).
