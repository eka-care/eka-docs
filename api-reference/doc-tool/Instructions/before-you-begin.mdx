This collection provides resources for integrating our API, enabling access to valuable healthcare management features and data.

## Prerequisites

Before you start using this collection, you need to complete the following steps:

1. Create your Eka developer account by following the steps presented in the Eka workspace page.
2. Obtain your `client_id` and `client_secret` for your business.
3. Gain access to resources like Appointment, Prescription, Doctor, etc., for your business.

## Getting Started

Once you have your developer account and credentials, you're ready to begin integrating with the Eka API. Here are some key resources to get you started:

1. [API Authentication Guide](#api-reference/doc-tool/auth/how-to-use-auth): Learn how to authenticate your requests and manage access tokens.
2. [API Endpoints](#api-reference/doc-tool/): Explore the available endpoints and their functionalities.
3. [Sample Integrations](#sample-integrations): View example integrations and code snippets.

## Authentication

Proper authentication is crucial for securely accessing the Eka API. Please refer to our detailed [API Authentication Guide](#api-reference/doc-tool/auth/how-to-use-auth) for step-by-step instructions on how to:

- Obtain an access token
- Use the access token in your API requests
- Handle token expiration and renewal

<CalloutBox type="info">
  Remember to keep your `client_id` and `client_secret` secure. Never share these credentials or expose them in client-side code.
</CalloutBox>
