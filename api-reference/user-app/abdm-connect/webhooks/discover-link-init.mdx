---
title: Link Initiation - Generate OTP
---

This webhook is triggered when a user initiates linking of discovered care contexts. Upon receiving this webhook, generate an OTP, send it to the user, and save the care contexts for further processing.

<Note>
After generating and sending the OTP, call the [On Link Init API](/api-reference/user-app/abdm-connect/care-contexts/discover/link_on_init) to notify Eka that the OTP has been dispatched.
</Note>
### Request
**Headers:**

- `Eka-Webhook-Signature`: `<Eka Webhook Signature>`  [Refer: Webhook Signature](/api-reference/connect/webhooks/webhook-signature)
- `Content-Type`: `application/json`
