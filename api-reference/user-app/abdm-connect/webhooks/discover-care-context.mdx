The webhook event is triggered whenever a user requests the discovery of unlinked care contexts. The HIP must use the provided user information to search for and identify matching unlinked
care contexts within their system.

### Request
**Headers:**

- `Eka-Webhook-Signature`: `<Eka Webhook Signature>`  [Refer: Webhook Signature](/api-reference/connect/webhooks/webhook-signature)
- `Content-Type`: `application/json`
