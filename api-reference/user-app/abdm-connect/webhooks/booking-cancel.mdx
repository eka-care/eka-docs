This Webhook is event is sent whenever the appointment booking is cancelled requested by this api
[Cancel Order](https://developer.eka.care/api-reference/user-app/abdm-connect/phys-cons/cancel)



### Request
**Headers:**

- `Eka-Webhook-Signature`: `<Eka Webhook Signature>`  [Refer: Webhook Signature](/api-reference/connect/webhooks/webhook-signature)
- `Content-Type`: `application/json`

**Body:**

```json
{
  "service": "uhi",
  "event": "booking_confirm",
  "event_time": 1749822309,
  "transaction_id": "<webhook transaction id>",
  "timestamp": 1749825214,
  "business_id": "<partner business id>",
  "client_id": "<partner client id>",
  "data": {
      "status": "<string>"
}
}
```
