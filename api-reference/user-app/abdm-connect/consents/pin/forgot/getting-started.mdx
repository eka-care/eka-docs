---
title: Overview
---


The user can initiate a PIN reset if they forget it, verify the process with an OTP,
and then reset the PIN, all via their mobile number.

<Note> User Authorization will be required for all these operations.</Note>


<Steps>
    <Step title="Initiate Password Reset: Forget Init">
        Use the [Forget Init](/api-reference/user-app/abdm-connect/consents/pin/forgot/forget-init) endpoint when a user
        forgets their PIN. This step initiates the reset process using the user’s mobile number.
    </Step>
    <Step title="Verify OTP: Forget Verify">
        Use the [Forget Verify](/api-reference/user-app/abdm-connect/consents/pin/forgot/forget-verify) endpoint to
        verify the OTP sent to the user’s mobile. This step is required to validate the OTP to reset PIN.
    </Step>
    <Step title="Reset PIN: Forgot Reset">
        Use the [Forgot Reset](/api-reference/user-app/abdm-connect/consents/pin/forgot/reset) endpoint to reset the
        PIN after successful OTP verification.
    </Step>
</Steps>
