---
openapi: POST /abdm/v1/ecdh/decrypt
title: Decrypt Data
description: >
  Decrypt data using ECDH key exchange.


  Use this API to decrypt health data received from external HIPs via ABDM. The decryption is performed using the ECDH key exchange mechanism.


  For details on key generation, encryption, and decryption, refer to the [ABDM Encryption and Decryption Guide](https://sandbox.abdm.gov.in/sandbox/v3/new-documentation?doc=EncryptionandDecryptions).
