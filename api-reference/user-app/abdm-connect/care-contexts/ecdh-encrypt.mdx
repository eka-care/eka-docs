---
openapi: POST /abdm/v1/ecdh/encrypt
title: Encrypt Data
description: >
  Encrypt data using ECDH key exchange.


  Use this API to encrypt health data before sharing it with external HIUs via ABDM. The encryption is performed using the ECDH key exchange mechanism.


  For details on key generation, encryption, and decryption, refer to the [ABDM Encryption and Decryption Guide](https://sandbox.abdm.gov.in/sandbox/v3/new-documentation?doc=EncryptionandDecryptions).
