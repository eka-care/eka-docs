---
openapi: GET /abdm/v1/ecdh/keys
title: Generate ECDH Key Pair
description: >
  Generate an ECDH key pair with nonce for encryption/decryption operations.


  This API generates a new ECDH key pair along with a nonce that can be used for encrypting and decrypting health data exchanged via ABDM.


  For details on key generation, encryption, and decryption, refer to the [ABDM Encryption and Decryption Guide](https://sandbox.abdm.gov.in/sandbox/v3/new-documentation?doc=EncryptionandDecryptions).
