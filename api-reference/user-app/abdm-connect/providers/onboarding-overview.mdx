---
title: Health Facility Onboarding
---
This API is used to onboard a healthcare facility into our system after it has been successfully registered on the ABDM portal and software linkage has been completed using the Eka Bridge ID.

## Prerequisites for Facility Onboarding
<Steps>
    <Step title="Create HPR ID">
       Visit https://nhpr.abdm.gov.in/createID to register as Facility Manager or Healthcare Professional
    </Step>
    <Step title="Register Facility (HFR ID)">
        Log in with HPR ID and add facility details, upload documents, and submit for approval. Once approved, you get the HFR ID
    </Step>
    <Step title="Software Linkage">
        Go to the facility dashboard, select Software Linkage, and use the Eka Bridge ID to register your facility under Eka Bridge.
    </Step>
</Steps>
