---
title: "Convert Notes"
openapi: "POST /voice/api/v1/transaction/{session_id}/convert-to-template"
---

Get the notes of a session in the format of any template — after the session has been processed. Use it to:

- **Convert** an existing session's notes into a different template (`template_id`),
- **Translate** the notes into another language (`target_language`),
- **Bring your own transcript** — pass `transcript` and EkaScribe structures *that* text with the template, instead of the session's stored transcript.

The call returns `202` with a `document_id`. Fetch the generated notes with [Get Document](/ekascribe/api-reference/documents/get-document) once its status is `success`.

<Tip>
A common flow: run a session with `clinical_notes_template`, then call Convert Notes with your custom `template_id` to get the same consultation in your own format — no re-recording needed.
</Tip>
