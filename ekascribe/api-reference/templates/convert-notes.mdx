---
title: "Convert Notes"
openapi: "POST /voice/api/v1/transaction/{session_id}/convert-to-template"
---

Get a session's notes in any template's format — convert to a different `template_id`, translate via `target_language`, or pass your own `transcript` to structure. Returns `202` with a `document_id` — fetch it with [Get Document](/ekascribe/api-reference/documents/get-document) once ready.
