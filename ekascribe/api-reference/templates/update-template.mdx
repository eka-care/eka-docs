---
title: "Update Template"
openapi: "PATCH /voice/api/v1/template/{template_id}"
---

Update a custom template's `title` or `desc` (the note-generation instructions). Only the fields you send are changed.

<Note>
Eka-provided default templates cannot be modified. Create your own with [Create Template](/ekascribe/api-reference/templates/create-template) instead.
</Note>
