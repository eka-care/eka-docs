---
title: "Get Document"
openapi: "GET /voice/api/v1/documents/{document_id}"
---

Fetch a document's metadata plus a presigned **GET** URL to download its markdown content. Use the `document_id` from [Get Session](/ekascribe/api-reference/sessions/get-session), [Convert Notes](/ekascribe/api-reference/templates/convert-notes) or [Regenerate Notes](/ekascribe/api-reference/sessions/regenerate-notes). Presigned URLs expire — fetch fresh, don't store.
