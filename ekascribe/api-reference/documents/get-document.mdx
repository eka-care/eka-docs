---
title: "Get Document"
openapi: "GET /voice/api/v1/documents/{document_id}"
---

Fetch a document's metadata and a **presigned GET URL** to download its content (markdown).

Use this to retrieve generated notes by `document_id` — the ID returned by [Get Session](/ekascribe/api-reference/sessions/get-session), [Regenerate Notes](/ekascribe/api-reference/sessions/regenerate-notes), [Convert Notes](/ekascribe/api-reference/templates/convert-notes) or [Create Document](/ekascribe/api-reference/documents/create-update-document).

```bash
# download the note content
curl -o note.md "<presigned_url>"
```

<Note>
Deleted (archived) documents return `404`. Presigned URLs expire — fetch a fresh one when needed rather than storing it.
</Note>
