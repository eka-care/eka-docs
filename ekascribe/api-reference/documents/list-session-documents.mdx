---
title: "List Session Documents"
openapi: "GET /voice/api/v1/sessions/{session_id}"
---

Fetch a session's full record, including the list of **all documents** on it — generated notes, transcripts and custom documents — with their status, template and timestamps.

Set `presigned=true` to also get a download URL for each document's content.

<Tip>
This is the listing surface for documents: one call per session returns everything generated on it. To fetch a single document by ID, use [Get Document](/ekascribe/api-reference/documents/get-document).
</Tip>
