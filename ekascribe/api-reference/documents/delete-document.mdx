---
title: "Delete Document"
openapi: "DELETE /voice/api/v1/documents/{document_id}"
---

Archive (soft-delete) a document. After deletion it disappears from [List Session Documents](/ekascribe/api-reference/documents/list-session-documents) and [Get Document](/ekascribe/api-reference/documents/get-document) returns `404` for it.

Deleting an already-deleted document is a no-op and still returns success.
