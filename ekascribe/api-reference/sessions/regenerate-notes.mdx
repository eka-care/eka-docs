---
title: "Regenerate Notes"
openapi: "POST /voice/v1/sessions/{session_id}/process/template/{template_id}"
---

Re-run note generation for an already-processed session with a specific template — for example to get a second output format from the same recording, or to apply a template you created after the consultation.

The call returns `202` immediately with a `document_id`. Retrieve the generated notes with [Get Document](/ekascribe/api-reference/documents/get-document) once its status is `success`, or poll [Get Session](/ekascribe/api-reference/sessions/get-session).

<Tip>
To convert notes using a transcript you already have (without a processed session recording), or to translate notes into another language, use [Convert Notes](/ekascribe/api-reference/templates/convert-notes) instead.
</Tip>
