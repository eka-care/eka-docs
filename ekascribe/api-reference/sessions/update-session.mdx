---
title: "Update Session"
openapi: "PATCH /voice/v1/sessions/{session_id}"
---

Update a session's metadata after creating it — attach or correct `patient_details`, change the requested `templates`, refine `language_hint`, or switch `session_mode` / `model`.

<Info>
`session_mode` and `model` can only be changed **before** [End Session](/ekascribe/api-reference/sessions/end-session) is called. After the session is committed, changing them returns `409`. Other fields (like `patient_details`) can still be updated.
</Info>

Only send the fields you want to change. Unknown fields are rejected with `422`.
