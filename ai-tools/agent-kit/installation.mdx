[Echo](https://github.com/eka-care/echo) is open-source and available on GitHub.

## Requirements

- Python 3.11+

## Install from PyPI

```bash
pip install echo-sdk
```

Or with uv:

```bash
uv pip install echo-sdk
```

## Install from GitHub

```bash
pip install git+https://github.com/eka-care/echo.git
```

## Verify

```python
import echo
print(f"Echo SDK version: {echo.__version__}")
```
